# jquery-sortable-lists
## jQuery plugin to sorting lists also the tree structures.

<p>You can sort an items of html lists by mouse. Create **tree structures**. Format all active items however you want. You can define the **isAlowed callback** which determines if dragged item can be inserted into another.
				Set the **insert zone** like a distance which determines if item will be inserted inside or outside of the active area, **speed of autoscroll**.</p>
<p>For more info look at <a href="http://localhost/jquery-sortable-lists/index.html">the documentation</a>.</p>
